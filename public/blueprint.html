07 Gargalos Identificados

Problemas que impedem o site de funcionar como funil de conversão completo.

Formulários sem backend

Alta

Assessment Wizard, Consultation Popup e Contact Form simulam envio com setTimeout. Nenhuma API route existe. Dados de leads são perdidos.

Impacto: Zero leads capturados. Todo o funil é visual mas não funcional.

ROI Calculator órfão

Alta

A página /tools/roi-calculator existe com calculadora funcional mas nenhum link no site aponta para ela. Navbar, footer, CTAs — zero referências.

Fix: Adicionar link no navbar, hero, FinalCTA e footer.

Sem tracking de conversão

Alta

Nenhum Meta Pixel ou Google Tag implementado no código. Ironia: oferecemos isso como serviço VH mas não usamos no próprio site.

Fix: Implementar no layout.tsx raiz.

Calendly não integrado

Alta

Página de contato menciona agendamento via Calendly mas é apenas placeholder. Nenhum widget real integrado.

Fix: Integrar Calendly embed ou usar cal.com.

Blog vazio

Média

Estrutura de blog existe (listing + MDX) mas zero posts publicados. Visitante chega via footer e vê “coming soon”.

Impacto: Zero conteúdo indexável para long-tail SEO. Perda de autoridade orgânica.

Páginas legais inexistentes

Média

Footer lista Privacy Policy, Terms of Service e Cookie Policy mas são textos estáticos sem link. Páginas não existem.

Impacto: Não-compliance com LGPD/GDPR. Risco legal para operação internacional.

Pricing sem fluxo diferenciado

Média

Os 4 planos VH (Starter/Growth/Premium/Master) apontam todos para o mesmo #audit. Não há parâmetro de plano no formulário.

Fix: Passar query param ?plan=growth e pré-selecionar no form.

Footer links de serviço genéricos

Média

Os 6 serviços no footer apontam todos para /services geral. Sem anchors para cada serviço específico.

Fix: Adicionar IDs por serviço na página e linkar com anchors.

Apenas 1 nicho no modal

Baixa

AI Marketing Modal mostra “Expertise de Nicho” com apenas Vacation Homes. Se há planos de expandir (restaurantes, clínicas), falta estrutura.

Oportunidade: Preparar para múltiplos nichos conforme o negócio escala.

404 page genérica

Baixa

Usa _not-found padrão do Next.js. Sem branding, sem CTAs úteis.

Fix: Criar 404 on-brand com links para Home e Contact.
